JavaScript

O JavaScript é uma linguagem de programação de alto nível voltada para o desenvolvimento web, criada originalmente para funcionar do lado do usuário, ou seja, nos navegadores. Junto do HTML e do CSS, é uma das principais tecnologias da web, permitindo a criação de páginas interativas com elementos dinâmicos e boa perfomance.

É uma linguagem multiparadigma (programação funcional e imperativa), versátil, com tipagem dinâmica (não é necessário definir os tipos das variáveis ao declará-las), sintaxe acessível, e recursos avançados como orientação a objetos e APIs para trabalhar com textos, matrizes, datas e expressões regulares. Atualmente, segue as especificações ECMAScript, que visam garantir o suporte e o funcionamento da linguagem entre os diferentes navegadores.

Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido, porém os mecanismos JavaScript agora estão incorporados em muitos outros tipos de software host, incluindo em servidores e bancos de dados da Web e em programas que não são da Web, como processadores de texto e PDF, e em tempo de execução ambientes que disponibilizam JavaScript para escrever aplicativos móveis e de desktop, incluindo widgets de área de trabalho.
